# Connectors

## How tool references work

solOS uses `~~category` as a placeholder for whatever tool the user connects in that category. The system is tool-agnostic: it describes workflows in terms of categories rather than specific products.

## Connectors for this plugin

| Category | Placeholder | Options |
|----------|-------------|---------|
| CRM | `~~crm` | HubSpot, Salesforce, Pipedrive, Close |
| Calendar | `~~calendar` | Google Calendar, Outlook Calendar |
| Email | `~~email` | Gmail, Outlook |
| Chat | `~~chat` | Slack, Microsoft Teams, Discord |
| Notes | `~~notes` | Notion, Obsidian, Confluence |
| Version Control | `~~git` | GitHub, GitLab |
